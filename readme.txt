HQLoginPlus is custom Login WP plugin. You can view the customized Wordpress login screen once you install this plugin.

== Upgrade Notice ==

This is the first version of HQLoginPlus plugin

== Description ==

To change the look of WordPress login page you can use this plugin.

== Installation ==

You can either install it automatically from the WordPress admin, or do it manually:

1. Unzip the archive and put the 'HQLoginPlus' folder into your plugins folder (/wp-content/plugins/).
1. Activate the plugin from the Plugins menu.

= Usage =

Once installed you can see the customized Wordpress login page 
Go to *WP-Admin -> Settings -> HQ Login* for configuration.
